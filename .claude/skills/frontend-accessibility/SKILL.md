# Frontend Accessibility

## When to use this skill:

- Creating accessible React, Vue, Angular, or Svelte components
- Adding ARIA labels, roles, and properties to interactive elements
- Implementing keyboard navigation and focus management
- Building accessible forms with proper labels and error handling
- Ensuring color contrast meets WCAG AA/AAA standards
- Adding alternative text for images and media
- Creating accessible navigation menus and modals
- Implementing skip links and landmark regions
- Testing with screen readers and accessibility tools
- Managing focus trap and keyboard interactions for modals/dropdowns

This Skill provides Claude Code with specific guidance on how to adhere to coding standards as they relate to how it should handle frontend accessibility.

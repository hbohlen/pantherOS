# Backend Queries

## When to use this skill:

- Writing SQL queries in .sql files or inline in code
- Creating ORM query methods (find, findOne, filter, aggregate)
- Implementing complex joins, subqueries, or window functions
- Adding database indexes for performance optimization
- Writing query builders or database abstraction layers
- Optimizing slow queries and analyzing query execution plans
- Handling pagination, sorting, and filtering in queries
- Implementing query caching strategies
- Managing database transactions and atomic operations
- Writing database views, stored procedures, or functions

This Skill provides Claude Code with specific guidance on how to adhere to coding standards as they relate to how it should handle backend queries.

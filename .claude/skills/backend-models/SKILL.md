# Backend Models

## When to use this skill:

- Defining new database models or entities in .js, .ts, .py, .go, .rb files
- Creating ORM models with Sequelize, Prisma, TypeORM, Django ORM, etc.
- Setting up model relationships (one-to-one, one-to-many, many-to-many)
- Adding validation rules and constraints to model fields
- Implementing model methods, hooks, or lifecycle callbacks
- Configuring database indexes and unique constraints
- Creating model factories or fixtures for testing
- Managing model versioning or migrations
- Setting up polymorphic associations or complex relationships

This Skill provides Claude Code with specific guidance on how to adhere to coding standards as they relate to how it should handle backend models.

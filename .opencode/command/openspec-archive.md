# /openspec-archive

Archive a completed OpenSpec change.

## Instructions
- Capture proposal name, link, status, and implementation summary.
- Summarize completed tasks, test results, and deployment targets.
- Document risks encountered, mitigations taken, and remaining gaps.
- Provide links to merged PRs, commits, and updated documentation.
- Create a concise changelog entry and follow-up backlog items.

## Output
- Archive report in markdown with sections for Summary, Evidence, Risks, Follow-ups.
- Checklist confirming proposal closed, artifacts linked, and docs updated.
- Recommendations for future improvements.

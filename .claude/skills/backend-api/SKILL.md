# Backend API

## When to use this skill:

- Designing new API endpoints or modifying existing ones in .js, .ts, .py, .go, .rb files
- Creating API controllers, middleware, or router configurations
- Implementing RESTful or GraphQL API structures
- Adding authentication and authorization logic to API routes
- Writing API documentation or OpenAPI/Swagger specifications
- Handling API requests/responses and HTTP status codes
- Building API gateway or proxy configurations
- Setting up API rate limiting, caching, or validation

This Skill provides Claude Code with specific guidance on how to adhere to coding standards as they relate to how it should handle backend API.

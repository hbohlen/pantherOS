# Global Validation

## When to use this skill:

- Validating form inputs on frontend (email format, password strength, etc.)
- Validating API request payloads and parameters
- Implementing schema validation with Joi, Yup, Zod, or other validators
- Type checking with TypeScript, Flow, or static analysis tools
- Validating database constraints and unique indexes
- Sanitizing user inputs to prevent injection attacks
- Validating file uploads (size, type, content)
- Implementing business rule validation
- Validating configuration files and environment variables
- Creating reusable validation functions and decorators
- Validating authentication and authorization logic

This Skill provides Claude Code with specific guidance on how to adhere to coding standards as they relate to how it should handle global validation.

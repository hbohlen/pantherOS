# /openspec-apply

Implement the change described in an approved OpenSpec.

## Instructions
- Summarize the source proposal (name, link, status) and key requirements.
- Break work into ordered tasks with owners, estimates, and dependencies.
- Align implementation with pantherOS module structure and host phases.
- Define test strategy (unit, integration, Nix builds) and acceptance criteria.
- Capture risks, mitigations, and rollback steps during execution.

## Output
- Present a live checklist of tasks with status.
- Record test commands and results.
- Provide deployment steps and rollback plan aligned to hosts/modules.
- List follow-up actions and documentation updates.

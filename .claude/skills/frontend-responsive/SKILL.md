# Frontend Responsive

## When to use this skill:

- Building mobile-first responsive layouts for websites
- Creating breakpoints for different screen sizes (mobile, tablet, desktop)
- Implementing responsive navigation menus and hamburger menus
- Making images, videos, and media responsive
- Using CSS Grid and Flexbox for responsive layouts
- Adapting typography and font sizes for different screens
- Handling mobile touch interactions and gestures
- Creating responsive forms that work on all devices
- Optimizing content density for small screens
- Testing responsive designs across different devices and browsers
- Implementing progressive enhancement for responsive features

This Skill provides Claude Code with specific guidance on how to adhere to coding standards as they relate to how it should handle frontend responsive.

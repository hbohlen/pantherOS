# Frontend CSS

## When to use this skill:

- Writing CSS, SCSS, SASS, or LESS stylesheets
- Using CSS-in-JS libraries like Styled Components, Emotion, or CSS Modules
- Implementing Tailwind CSS utility classes
- Creating CSS animations and transitions
- Building layouts with Flexbox and CSS Grid
- Designing responsive breakpoints and media queries
- Implementing dark mode or theme switching with CSS variables
- Creating component-specific styling and avoiding global conflicts
- Managing CSS specificity and avoiding !important overuse
- Setting up CSS architecture like BEM, OOCSS, or ITCSS methodologies

This Skill provides Claude Code with specific guidance on how to adhere to coding standards as they relate to how it should handle frontend CSS.

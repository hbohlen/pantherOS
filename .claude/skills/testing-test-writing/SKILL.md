# Testing Test Writing

## When to use this skill:

- Writing unit tests for functions, components, and modules
- Creating integration tests for API endpoints and database operations
- Building end-to-end tests with Cypress, Playwright, or Selenium
- Setting up test suites with Jest, Mocha, Jasmine, or pytest
- Writing test fixtures and factory functions
- Mocking external dependencies and services
- Creating test coverage reports and increasing coverage
- Testing React components with React Testing Library
- Writing contract tests and API schema validation tests
- Setting up test databases and test environments
- Implementing snapshot testing for UI components

This Skill provides Claude Code with specific guidance on how to adhere to coding standards as they relate to how it should handle testing test writing.

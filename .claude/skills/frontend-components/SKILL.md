# Frontend Components

## When to use this skill:

- Creating new React, Vue, Angular, or Svelte components
- Building reusable component libraries or design systems
- Implementing component composition patterns
- Managing component state and lifecycle methods
- Creating higher-order components or custom hooks
- Building form components with controlled/uncontrolled inputs
- Implementing component prop validation and TypeScript interfaces
- Creating modal, dropdown, or overlay components
- Building data visualization or chart components
- Developing button, input, card, or layout components

This Skill provides Claude Code with specific guidance on how to adhere to coding standards as they relate to how it should handle frontend components.
